# Иван · Fullstack-разработчик

**Сайт-визитка:** [h1nigami.github.io/about](https://h1nigami.github.io/about/)

Пишу бэкенд на **Go**, **Python** и **C#**, фронтенд на **TypeScript**
и **React**, десктоп — на **WPF** и **Avalonia**. Специализируюсь на

и транспортная телематика.

## Чем занимаюсь

- 🦺 **Видеоаналитика СИЗ** — детекция средств индивидуальной защиты
  на видеопотоке в реальном времени (обучение моделей, OpenCV, инференс)
- 🗺️ **Собственный геокодер** — адрес ⇄ координаты без внешних платных API
- 🏭 **Операционная платформа для предприятий** — транспорт, персонал
  и видеонаблюдение в одном контуре (в активной разработке)
- 📡 **Телематика** — протоколы JT808 / JT1078 / CMSV6 на уровне
  бинарных пакетов: телеметрия, видео с бортовых камер

**Инструменты:** Kafka · Redis · TimescaleDB · PostgreSQL · Docker · gRPC ·
ASP.NET Core · Entity Framework · Dapper · CI/CD

## О репозитории

Сайт — один файл `index.html` без зависимостей и сборки: canvas-частицы,
рамки детекции в стиле CV, HUD с живым логом, scroll-анимации.
Локально: просто открыть `index.html` в браузере.

Деплой автоматический: пуш в `main` → GitHub Actions
([deploy.yml](.github/workflows/deploy.yml)) публикует сайт в ветку
`gh-pages` → GitHub Pages.
